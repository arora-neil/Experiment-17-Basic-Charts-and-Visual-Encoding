# 3. Concepts Used

1. Data Visualization
   
    Data visualization is the graphical representation of data using charts and plots to make information easier to understand.

3. Visual Encoding
   
    Visual encoding refers to representing data using visual elements such as:
    - Position (x-axis, y-axis)  
    - Color  
    - Size
    - Shape  

3. Types of Charts
  - Line Chart  
  - Bar Chart  
  - Histogram  
  - Scatter Plot  

4. Libraries Used
- Matplotlib  
- Seaborn  

# Theory

1. **Introduction to Data Visualization**
   
  Data visualization is an essential part of exploratory data analysis. It helps in identifying patterns, trends, and relationships within the data. Visual representation makes complex data easier to understand compared to raw numerical values.

2. **Line Chart**
   
A line chart is used to show the relationship between two variables and is especially useful for showing trends over time.

Characteristics:
- Data points are connected using lines  
- Shows trends and changes  
- Useful for continuous data  

3. **Bar Chart**
   
A bar chart represents data using rectangular bars where the height of each bar corresponds to the value.

Characteristics:
- Used for categorical data  
- Easy to compare values  
- Can be vertical or horizontal  

4. **Histogram**
   
A histogram is used to show the distribution of numerical data.

Characteristics:
- Divides data into intervals (bins)  
- Shows frequency of values  

5. **Scatter Plot**
   
A scatter plot represents the relationship between two variables using points.

Characteristics:
- Shows correlation between variables  
- Each point represents a data pair  

6. **Visual Encoding Techniques**
   
Visual encoding improves the clarity and effectiveness of graphs.

Types of Encoding:

  * Position - Determines placement of data on axes  

   *  Color - Used to differentiate categories  

  * Size - Represents magnitude  

  * Shape - Different markers represent different data points  

7. **Common Color Options**
